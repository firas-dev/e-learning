Conception et Développement d’une Plateforme E-Learning 
Intégrant un Module de Détection des Émotions à travers les expressions Faciales 

Ce projet consiste en la conception et le développement d’une plateforme E-Learning intelligente 
intégrant un module d’intelligence artificielle dédié à la détection des émotions des apprenants 
à travers l’analyse de leurs expressions faciales.
L’objectif principal de ce projet est d’améliorer l’efficacité de l’enseignement en ligne en intégrant 
une dimension affective au processus d’apprentissage


This project is a full-stack web application built using:

- Frontend: React (Vite)
- Backend: Django REST Framework
- Database: MongoDB

---

- React Router

- MongoDB
